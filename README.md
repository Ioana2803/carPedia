# 🚗 CarPedia

A multi-page automotive encyclopedia website built with HTML, CSS, and JavaScript.
Created as a personal project combining a passion for cars with web development skills.

## 🔗 Live Demo
[View CarPedia](https://ioana2803.github.io/carPedia/)

## 📸 Pages & Features

### 🏠 Home
- Hero section with full-width image
- General history of the automobile
- Search bar to find car brands directly

### 🏷️ Brands
- Dropdown navigation menu with all brands
- Card grid with image and description for each brand:
  Aston Martin, Audi, BMW, Bugatti, Dodge, Ferrari,
  Jaguar, Lamborghini, McLaren, Mercedes, Nissan,
  Pagani, Porsche, Rolls-Royce, Toyota, Volkswagen

### 🚘 Car Details
- Individual page for each brand
- Full-width hero image with brand name
- Brand history section
- Additional info cards with images

### 📰 News
- Fetches live automotive news via external API
- Fully functional when run locally with a valid API key

### 💬 Community
- Users can post and delete messages
- Custom username support
- Designed to connect with a backend/database

### 🏛️ Museums
- Cards with photos, location, and description
- Search by museum name or country
- Direct link to each museum's official website

### ℹ️ About
- Project introduction and motivation
- Technologies used
- Developer profile

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
- External News API (local only)

## 🚀 Run Locally
```bash
git clone https://github.com/Ioana2803/carPedia.git
cd carPedia
```
Open `index.html` in your browser.

> ⚠️ The News page requires a valid API key to function locally.
